## Current Projects

### FASTA Sequence Analysis

A beginner bioinformatics project focused on working with DNA sequences stored in FASTA format.

Current functionality:

* Read FASTA files using Biopython.
* Extract sequence information.
* Count nucleotide frequencies (A, T, G, C).
* Handle uppercase and lowercase DNA sequences.
* Explore basic sequence manipulation workflows.

## Installation

Clone the repository:

```bash
git clone https://github.com/oscarflores300107-design/bioinformatics-portfolio.git
```

Create and activate the environment:

```bash
conda create -n bioinfo python=3.12
conda activate bioinfo
```

Install dependencies:

```bash
pip install biopython
```
